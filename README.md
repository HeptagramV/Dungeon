个人尝试向作品

开发工具：IDEA

开发时间：2021年

游戏类型：类rogue文字回合制地牢

大一时期可以说是第一次自己对游戏设计的尝试，当时完全不会用引擎，纯java代码手撸了一个小小的文字冒险游戏，稍微带有一些rogue元素
